# 🏛️ National Translation Hub 

### A Multilingual AI Assistant for Citizen Services

The *National Translation Hub* is a multidisciplinary AI-powered web application designed to bridge the language gap in India. It enables users to communicate across *15+ Indian languages* and *5 global languages* using real-time Voice-to-Text and Text-to-Voice technology.

---

## 🚀 Features
* *Speech-to-Text (STT):* Real-time voice recognition for Kannada, Hindi, Bengali, and more.
* *Neural Machine Translation (NMT):* High-accuracy translation using Google’s Neural engine.
* *Text-to-Speech (TTS):* Instant audio playback in the target language.
* *Professional UI:* Clean, government-inspired interface built with Tailwind CSS.
* *Global Reach:* Includes support for English, French, German, Japanese, and Spanish.

---

## 🛠️ Tech Stack
* *Backend:* Python 3.12, Flask
* *AI Libraries:* googletrans, gTTS
* *Frontend:* HTML5, Tailwind CSS, JavaScript (Web Speech API)
* *Server:* Localhost (WSGI)
